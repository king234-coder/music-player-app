# 🎵 Music Player App

A stylish and interactive music player web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ▶️ Play / Pause music
- ⏭ Next / Previous track
- 🎚 Progress bar control
- 🔊 Volume control
- 🎧 Clean and responsive UI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

## 📂 Project Structure

```
music-player-app/
│── index.html
│── style.css
│── script.js
│── /songs
│── /images
```

## 💡 Future Improvements
